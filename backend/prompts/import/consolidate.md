## 任务

1. **去重**：识别语义相同或高度相似的题目，合并为一条，保留最完整的答案
2. **分类**：将题目按主题分组（如"基础概念"、"并发编程"、"网络协议"等），每组至少2题，不足2题的归入"综合"分类
3. **统一格式**：确保每道题的答案完整、格式统一
4. **补充标签**：为每道题补充或优化关键词标签（3-5个）

## 输入

{{questions}}

## 输出要求

严格输出以下 JSON 格式，不要输出任何其他内容，不要用 markdown 代码块包裹：

{"categories":[{"name":"分类名称","items":[{"q":"题目","a":"参考答案","k":["关键词1","关键词2"]}]}],"dedupCount":去重合并的题目数量}

注意：
- dedupCount 是被合并/去重掉的题目数量（原始总数 - 清洗后总数）

- 答案保留原始内容，仅在格式不统一时进行整理
- 所有内容使用中文
