你是一位内容整理助手。现在有一批从网页抓取的面试题，请你进行清洗整理。

## 任务

1. **去重**：识别语义相同或高度相似的题目，合并为一条，保留最完整的解答
2. **分类**：将题目按主题分组（如"基础概念"、"并发编程"、"网络协议"等），每组至少2题，不足2题的归入"综合"分类
3. **格式统一**：确保题目和解答的格式清晰明了

## 输入

{{questions}}

## 输出要求

严格输出以下 JSON 格式，不要输出任何其他内容，不要用 markdown 代码块包裹：

{"categories":[{"name":"分类名称","items":[{"q":"题目","a":"解答"}]}],"dedupCount":去重合并的题目数量}

注意：
- dedupCount 是被合并/去重掉的题目数量（原始总数 - 清洗后总数）
- 如果只有少量题目无法形成有意义的分类，全部归入"综合"
- 保留原始题目和解答内容，仅做格式整理
- 不要自行生成答案，不要添加关键词
- 所有内容使用中文
