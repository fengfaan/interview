待分析简历内容：

{{resume}}
"""

逐段扫描维度：
1. **模块完整性**：是否缺少联系方式、教育背景、工作/项目经历、专业技能等必要模块
2. **排序逻辑**：经历是否倒序排列，非应届生是否工作经历优先于教育背景
3. **篇幅密度**：自我评价是否过长挤占核心经历篇幅、非核心项目是否堆砌、核心经历是否描述过短
4. **表达质量**：是否有口语化/模糊/空洞描述（如"负责了系统开发"、"参与了项目"）、是否缺少数据指标、动词是否软弱
5. **冗余内容**：是否有重复描述、无关信息、过时的技能标签、无价值的自我评价

请以严格的 JSON 格式输出，不带 Markdown 块：
