## MODIFIED Requirements

### Requirement: 批量导入多道题目
系统 SHALL 支持一次导入多道面试题到 Obsidian 知识库，支持两种模式：逐条保存（原有）和合并保存（新增）。逐条模式保存每条独立汇报结果；合并模式将所有题目写入单个 Markdown 文件。

#### Scenario: 批量导入成功（逐条模式）
- **WHEN** 用户从网页解析结果中选择多道题目点击"导入到知识库"
- **THEN** 系统逐条保存，每条保存成功后更新该条的状态为"已保存"

#### Scenario: 批量导入部分失败（逐条模式）
- **WHEN** 批量导入中某些题目保存失败
- **THEN** 系统继续保存剩余题目，完成后汇报成功和失败的数量

#### Scenario: 合并保存成功
- **WHEN** 用户从网页解析结果中选择多道题目点击"AI 清洗并合并保存"
- **THEN** 系统先执行 AI 清洗，然后将清洗后的所有题目合并写入单个 Markdown 文件，返回文件路径和题目数量
